## Query 1

```sql
-- @__p_0='3'
SELECT (
    SELECT count(*)::int
    FROM "Products" AS p0) AS "TotalCount", p."Name", p."BrandId", p."Id"
FROM "Products" AS p
ORDER BY p."Name" DESC, p."Id"
LIMIT @__p_0
```

## Query 2

```sql
-- @__ids_0={ '2', '5' } (DbType = Object)
```
