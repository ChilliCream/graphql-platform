---
title: logout Command
description: "The `nitro logout` command ends your Nitro CLI session and removes locally stored credentials; afterwards commands need `nitro login` or an `--api-key`."
---

The `nitro logout` command logs you out and removes the locally stored session information. After logout, subsequent commands either need a fresh [`nitro login`](./login.md) or an explicit `--api-key`.

```shell
nitro logout
```
